# A Machine-Learning-Based Integrative Framework Utilizing Mendelian Randomization
This repository contains all original source code, machine learning pipelines and Mendelian randomization analysis scripts for the manuscript submitted to iScience.

## Operating Environment & Dependencies
- Python 3.8+ / R 4.1+
- All required packages are listed in the supplementary configuration file.

## Reproduction Guide
1. Install dependent software packages following the configuration file.
2. Run the main script to complete MR causal inference and machine learning model training.
3. Scripts for result visualization and statistical plotting are provided in the plotting folder.

## Access Statement
This repository is fully public. No login, password or personal identity authentication is required for all reviewers and readers to view, download and run the complete source code. All computational results and figures in the manuscript can be fully reproduced via the code in this repository.
